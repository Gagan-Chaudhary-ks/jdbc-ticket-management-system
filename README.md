# 🎟 Ticket Management System (WIP)

## 📌 Overview

A backend system for managing user tickets and support requests using **Java, JDBC, and MySQL**.

> 🚧 This project is currently under development.

---

## 🚀 Current Features

* User creation and management
* Ticket creation functionality
* Basic database connectivity using JDBC

---

## 🛠 Tech Stack

* Java
* JDBC
* MySQL

---

* Improved error handling & validation
* Service-layer architecture refinement
* Transaction management (commit/rollback)
